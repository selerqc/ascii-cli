# ascii-cli

A command-line tool to display ASCII art logos, inspired by neofetch.

## Commands

Once the application is running, you can use the following commands:

*   `ascii --show <name>`: Show a specific ASCII art.
    *   Example: `ascii --show arch`
*   `ascii --list-logos` or `--ll`: List all available logos.
*   `ascii --print-logos` or `--pl`: Print all available logos.
*   `clear` or `c`: Clear the terminal.
*   `help`: Show all available commands.
*   `exit`: Exit the application.
